# Yeşilİz

Yeşilİz, kullanıcıların geri dönüşüm faaliyetlerini takip ederek puan kazandığı ve çevreye katkılarını görebildiği bir masaüstü uygulamasıdır.

## Özellikler

- Kullanıcı kayıt ve giriş sistemi
- Geri dönüşüm puanlama sistemi
- Tohum puanı ve fidan sistemi
- Liderlik tablosu
- Geri dönüşüm istatistikleri
- Karbon tasarrufu hesaplama
- Etkinlik takibi
- SQLite veritabanı

## Teknolojiler

- Python
- PyQt5
- SQLite

## Geliştiriciler

- Kübra Belsizoğlu
